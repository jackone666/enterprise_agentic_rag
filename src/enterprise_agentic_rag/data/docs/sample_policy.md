# 企业知识库使用政策

## 1. 文档访问权限
- 所有员工可访问公开知识库文章
- 部门内部文档仅限本部门成员访问
- 机密文档需要额外审批

## 2. 数据分类标准
| 级别 | 标签 | 说明 |
|------|------|------|
| L1 | 公开 | 可对外分享 |
| L2 | 内部 | 仅限公司内部 |
| L3 | 机密 | 需审批访问 |
| L4 | 绝密 | 最高安全级别 |

## 3. 信息保密要求
- 不得在回答中泄露用户个人信息
- 不得透露内部系统架构细节
- 涉及财务数据需脱敏处理

## 4. 回答质量标准
- 所有回答必须引用来源
- 不确定的信息必须明确标注
- 涉及安全操作需要人工确认

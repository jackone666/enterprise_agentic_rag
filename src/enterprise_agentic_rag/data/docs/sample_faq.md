# 常见问题 FAQ

## Q: 如何重置密码？
访问 https://account.company.com/reset，输入企业邮箱，系统发送重置链接，有效期 30 分钟。

## Q: 如何申请 API 访问密钥？
登录管理控制台 → 开发者设置 → API 密钥 → 创建新密钥。需要直属上级审批。

## Q: 工单处理时效是什么？
紧急 2 小时、高优先级 4 小时、普通 1 个工作日、低优先级 2 个工作日。

## Q: SDK 接入报 AUTH_401 错误怎么办？
检查 API Key 是否过期、确认请求头正确传递 Key、验证权限范围、重新生成 Key、联系管理员检查 IP 白名单。

## Q: 遇到系统故障如何上报？
检查系统状态页面确认是否为已知问题，如为新问题创建紧急工单，在工作群通知运维团队，保留日志截图。
